/* client.c READ ME */

1. Compile the program using
	gcc -o client client.c -lm

2. Run the program, providing the file of client data as a command line argument, e.g.
	./client input-data.txt

3. When prompted, enter the integer command for what you would like to search 
	0 = exit
	1 = first name
	2 = last name
	3 = phone number
	4 = email address

4. At the next prompt, enter the value to search for (corresponding to the integer command above)

5. Enter a command of 0 to exit the program
